Gestione di una Rubrica Telefonica
Questo progetto consiste in un'applicazione Java testuale che si connette a un database MySQL per gestire i dati di una rubrica telefonica. L'applicazione offre funzionalità CRUD (Create, Read, Update, Delete) per i contatti della rubrica

Requisiti
Java Development Kit (JDK) versione 8 o successiva
MySQL Server
Librerie JDBC per MySQL (solitamente fornite dal driver MySQL)

Utilizzo
Una volta avviata l'applicazione, sarà possibile:

Aggiungere, visualizzare, modificare ed eliminare contatti dalla rubrica
Inserire manualmente i dati richiesti, come nome contatto, cognome, numero di telefono, email
Le operazioni di modifica ed eliminazione richiederanno l'inserimento dell'ID del record corrispondente.
